# Day 14: Higher or Lower Game

A simplified version of the Higher or Lower game where the player guesses which celebrity has more Instagram followers.

## How to play

1. Run the script and type `Y` to start
2. You are presented with two celebrities, **A** and **B**
3. Guess which one has more Instagram followers by typing `A` or `B`
4. If you're right, the game continues — celebrity B becomes the new A, and a new B is picked
5. If you're wrong, the game ends and shows your final score
6. After each game you can choose to play again or quit

## Tech

- Python 3
- No external libraries required
